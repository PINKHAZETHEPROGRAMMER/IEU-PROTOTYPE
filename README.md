Nombre del proyecto:
INFOFUT-Pagina Web sobre el MUNDIAL.

Propósito:
Prototipo funcional que tiene como proposito ofrecer una propuesta para una pagina web sobre el mundial.

Contenido del repositorio
34 carpetas con su respectivo HTML y CSS, una carpeta con informacion global y una carpeta de Assets

Características principales:

Implementacion de responsividad, dashboard dinámico con posts, sistema de usuarios (admin / usuario normal), microinteracciones visuales en botones y accesibilidad basica.
